# Video Processing API

FastAPI приложение для обработки видео в реальном времени с использованием OpenCV. Предоставляет REST API и WebSocket для потоковой передачи видео с веб-камеры с возможностью детекции лиц и применения фильтров.

## 📋 Возможности

- 📹 Потоковое видео с веб-камеры (MJPEG stream)
- 🔍 Детекция лиц в реальном времени
- 🎨 Применение фильтров (градации серого, выделение границ)
- 🌐 WebSocket соединение для двусторонней передачи видео
- 🎛 Управление камерой через REST API
- 📊 Информация о доступных камерах
- 🔧 Автоматическое освобождение ресурсов

## 🛠 Технологии

- **FastAPI** - современный веб-фреймворк
- **OpenCV** - компьютерное зрение
- **WebSocket** - реальное время
- **Uvicorn** - ASGI сервер

## 🚀 Установка и запуск

### pip install fastapi uvicorn opencv-python numpy
### uvicorn video_processing:app --reload --host 0.0.0.0 --port 8000


## 📚 API Эндпоинты

### Видео поток
GET	/video-feed	MJPEG поток с веб-камеры с детекцией лиц

### Управление камерой
**GET	/stop-camera** - Остановить камеру и освободить ресурсы

**GET	/camera-status** - Проверить статус камеры

**GET	/camera-info**	- Информация о доступных камерах

### Системные
**GET	/health**	- Проверка здоровья сервера

**POST	/shutdown**	- Завершение работы сервера
**WebSocket	/ws/video**	- WebSocket для потокового видео

### Документация
**GET	/docs**	- Интерактивная документация Swagger UI

**GET	/redoc**	- Альтернативная документация ReDoc
